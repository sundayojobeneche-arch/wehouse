# WeHouse - Student Housing Platform v2

A full-stack student housing web application built with React, TypeScript, Tailwind CSS, and Supabase.

## Features

- Student & Staff authentication with role-based access
- Hostel search, filter, and booking system
- Paystack payment integration with installment plans
- Stay tracking with countdown and alerts
- Roommate matching system with chat
- Admin dashboard for managing hostels, bookings, payments
- Staff dashboard for managing assigned properties

## Tech Stack

- React 19 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Supabase (Auth, Database, Storage)
- Recharts (Analytics)

## Getting Started

1. Clone the repo
2. Run `npm install`
3. Create `.env` file with Supabase credentials
4. Run `npm run dev`
